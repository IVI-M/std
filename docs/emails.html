AMA Documents show Dosages Varied by 20 times in Concentration

This document shows that vaccines were alotted special codes to identify dosage levels.

  • Vaccines with code 91301 had a dosage of 200 micrograms per ml
  • Vaccines with code 91300 had a dosage of 100 micrograms per ml
  • Vaccines with code 91304 had a dosage of 10 micrograms per ml
  • Vaccines with code 91305 had a dosage of 100 micrograms per ml
  • Vaccines with code 91306 had a dosage of 200 micrograms per ml
  • Vaccines with code 91307 had a dosage of 50 micrograms per ml
  • Vaccines with code 91308 had a dosage of 15 micrograms per ml
  • Vaccines with code 91309 had a dosage of 100 micrograms per ml

It is therefore obvious that some people were receiving a dosage 20 times higher than others. The AMA document makes iot clear that these different dosages were targetted to different age groups - babies, children, teens, adults. The codes are not vax lot numbers, they are ICD billing codes, and different dose levels are just different vials and doses for age groups.
